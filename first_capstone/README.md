
# 🎧 سرد (Sard)

**سرد** هو تطبيق بودكاست بسيط وسلس يركّز على تجربة الاستماع بدون تعقيد، وبواجهة نظيفة وسهلة الاستخدام.

---

## 📱 فكرة التطبيق

تطبيق سرد يسمح لك تستمع للحلقات الصوتية بطريقة مرتبة وسهلة، مع مشغل صوت مدمج وتجربة استخدام مريحة.
[https://drive.google.com/file/d/1KdCGxZ9Zg34pRaWA7fCtdEa4m8UsIQJa/view?usp=drive_link
## ✨ المميزات

- 🎙️ تشغيل حلقات البودكاست  
- ⏯️ تحكم كامل بالصوت (تشغيل، إيقاف، تقديم، تأخير)  
- 📚 عرض البودكاست والحلقات  
- 🎧 مشغل صوت داخل التطبيق  
- 🌙 واجهة بسيطة وخفيفة  

---

## 🏗️ هيكل المشروع

التطبيق مبني باستخدام:

- Flutter  
- BLoC / Cubit لإدارة الحالة  
- Repository Pattern  
- Service Locator
